# Personal AI Employee Hackathon Project

Welcome to the Personal AI Employee project! This repository contains the implementation of a Bronze Tier AI Employee system as part of the hackathon.

## Project Structure

All project files are organized in the `Vault` directory following the Obsidian-style structure:

```
Vault/
├── Dashboard.md              # Central monitoring dashboard
├── Company_Handbook.md       # Rules and guidelines for AI employee
├── Business_Goals.md         # Business objectives and metrics
├── Claude_Code_Interaction.md # Claude Code integration example
├── BRONZE_TIER_COMPLETE.md   # Completion verification
├── agent_skills.md           # Agent skills documentation
├── filesystem_watcher.py     # File system monitoring
├── orchestrator.py           # Task coordination system
├── requirements.txt          # Python dependencies
├── README.md                 # Internal project documentation
├── Inbox/                    # Incoming tasks
├── Needs_Action/             # Pending tasks for processing
├── Done/                     # Completed tasks
├── Plans/                    # Processing plans
├── Pending_Approval/         # Tasks requiring human approval
└── Logs/                     # System logs
```

## Quick Start

All development and operations should be performed from within the `Vault` directory:

```bash
cd Vault
pip install -r requirements.txt
python orchestrator.py
```
